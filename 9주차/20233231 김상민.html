전산기초실습 9주차 실습과제

위의 제목(전산기초실습 9주차 실습과제)은 <h1> 태그에 style 속성을 사용하여 글자색과 배경색을 지정하세요. 페이지 하단의 학번과 이름 부분은 <p> 태그에 style 속성을 사용하여 기울이기와 글자색, 글자 크기를 지정하세요. (인라인 스타일 시트 사용하기)

본 내용 부분은 <style> 태그 내에 class 셀렉터를 사용하여 글자색과 글자 크기, 배경색을 지정하세요. 가상 클래스(pesudo class)를 사용하여 첫 줄의 글자색과 첫 글자의 크기를 지정하고, 마우스 포인터를 올리면 배경색이 다른 색으로 바뀌도록 지정하세요(내부 스타일 시트 사용하기).

글자 크기는 font-size, 글자색은 color, 배경색은 background 속성을 사용하고, 기울이기는 font style: italic;을 사용하세요. 글자색과 배경색, 글자 크기는 자유롭게 지정하세요.

외부 스타일 시트 사용하기

이 부분은 외부 스타일 시트를 사용하세요. 글자 크기와 글자색, 배경색은 id 셀렉터를 사용하여 자유롭게 지정하세요.
